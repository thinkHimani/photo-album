# Intelligent Photo Album
Implemented a photo album web application, that can be searched using natural language
through both text and voice

### Services/ Technologies:
- AWS S3
- AWS API Gateway
- Swagger
- AWS Lambda
- AWS Lex
- AWS Rekognition
- DynamoDB
- AWS Opensearch/ElasticSearch
- AWS Cloudwatch
- AWS CodePipeline
- AWS CloudFormation
